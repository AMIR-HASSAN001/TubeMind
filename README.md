# TubeMind

TubeMind is an AI-powered YouTube RAG Chatbot built using FastAPI, LangChain, FAISS, Hugging Face Embeddings, and Groq.

## Features

- 🎥 Analyze YouTube videos using transcripts
- 💬 Ask questions about the video
- ⚡ FastAPI backend
- 🧠 LangChain + FAISS Retrieval
- 🤖 Groq LLM
- 🔍 Semantic Search using Hugging Face Embeddings
